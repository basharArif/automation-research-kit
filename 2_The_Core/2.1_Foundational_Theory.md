## Control Systems: Principles & Mathematics

**Negative Feedback Loop**: The core principle of automation. System generates output \(y(t)\); desired value is reference \(r(t)\); error is \(e(t) = r(t) - y(t)\); controller produces control input \(u(t)\) to reduce error. Negative feedback (subtracting actual from desired) stabilizes behavior.

**Mathematical Representation**: State-space form describes any linear system:

\[\dot{x}(t) = Ax(t) + Bu(t)\]
\[y(t) = Cx(t) + Du(t)\]

where \(x \in \mathbb{R}^n\) is state, \(u \in \mathbb{R}^m\) is control input, \(y \in \mathbb{R}^p\) is output, and matrices \(A, B, C, D\) define system dynamics.

**Transfer Function** (Laplace domain): For linear systems, transfer function \(G(s) = Y(s)/U(s)\) captures input-output relationships in frequency domain, enabling classical control design (Bode plots, root locus, Nyquist criteria).

**Stability Analysis**:
- **Lyapunov stability**: System is stable if all trajectories remain bounded or converge to equilibrium
- **Eigenvalues**: For linear systems, stability ⟺ all eigenvalues of \(A\) have negative real parts
- **Routh-Hurwitz criterion**: Algebraic test for polynomial roots without computing them
- **Frequency domain**: Nyquist plot encircles \(-1\) point ⟺ system unstable

**PID Control Tuning**: Three classical methods:
1. **Ziegler-Nichols**: Experimentally induce oscillation at stability boundary, derive tuning formulas
2. **Cohen-Coon**: Improved formulas based on process reaction curve
3. **Loop shaping**: Design frequency response to meet bandwidth and stability margin specifications

## Automata Theory & State Machines

**Finite State Machines (FSM)**: Abstract model with discrete states and transitions. Every PLC program and most digital logic is fundamentally an FSM.

**Turing Completeness**: Universal computing machines (Turing, 1936) can compute any computable function. PLCs and microcontrollers are Turing-complete, enabling arbitrary logic.

## Cybernetics & Feedback

**Homeostasis**: Biological systems maintain internal equilibrium despite external perturbations through feedback loops. Applies to physical automation—maintaining setpoint despite load changes.

**Negative vs Positive Feedback**:
- **Negative**: Error reduces over time; stabilizing (Watt's governor)
- **Positive**: Error amplifies; destabilizing (audio feedback screech)
