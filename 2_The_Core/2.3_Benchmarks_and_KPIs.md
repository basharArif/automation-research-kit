# Performance Metrics & KPIs for Automation

**Version**: 1.0 | **Date**: October 25, 2025

---

## 1. Overview

Benchmarks and Key Performance Indicators (KPIs) are essential for evaluating the performance of automation systems and for driving continuous improvement. This document provides a summary of the most important KPIs for the key sectors of automation.

## 2. Industrial Automation KPIs

*   **Overall Equipment Effectiveness (OEE):** A measure of how well a manufacturing operation is utilized compared to its full potential. OEE is calculated as: `OEE = Availability x Performance x Quality`.
    *   **Availability:** The percentage of scheduled time that the system is available to operate. `Availability = Run Time / Planned Production Time`.
    *   **Performance:** The speed at which the machine runs as a percentage of its designed speed. `Performance = (Ideal Cycle Time × Total Count) / Run Time`.
    *   **Quality:** The percentage of good parts produced out of the total parts produced. `Quality = Good Count / Total Count`.
*   **Mean Time Between Failures (MTBF):** The average time between system breakdowns. `MTBF = Total Uptime / Number of Breakdowns`.
*   **Mean Time To Repair (MTTR):** The average time it takes to repair a system after a failure. `MTTR = Total Downtime / Number of Breakdowns`.

## 3. Robotics & Autonomous Systems KPIs

*   **Success Rate:** The percentage of tasks that are completed successfully.
*   **Latency:** The time it takes for a system to respond to a stimulus. In robotics, this is often measured as the time from sensor input to actuator output.
*   **Sim2Real Transfer Gap:** A measure of the performance difference between a policy trained in simulation and the same policy deployed on a real robot. `Gap = (Performance_sim - Performance_real) / Performance_sim`.

## 4. Robotic Process Automation (RPA) KPIs

*   **Time Savings:** The amount of time that is saved by automating a process.
*   **Error Rate:** The percentage of errors that occur in an automated process, compared to the manual process.
*   **Cost Savings:** The amount of money that is saved by automating a process.
*   **Process Velocity:** The increase in the speed at which a process is completed.
