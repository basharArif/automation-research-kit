# Automation Unlocked

A complete research and learning kit for automation, covering foundational principles from classical PID control to modern AI-driven systems. Includes detailed documentation and hands-on Python experiments.

**Version**: 1.0  
**Last Updated**: 2026-05-07  
**License**: MIT

## A Narrative-Driven Approach

This repository is structured as a narrative, guiding you through the world of automation in a logical and intuitive way. The project is divided into four main sections:

*   **[1_The_Genesis/](1_The_Genesis/)**: Where automation came from (Introduction, History).
*   **[2_The_Core/](2_The_Core/)**: The fundamental principles and mechanics (Theory, Core Experiments).
*   **[3_The_Applications/](3_The_Applications/)**: How automation is used in the real world (Sectors, Applied Experiments).
*   **[4_The_Horizons/](4_The_Horizons/)**: Where automation is going (Future Research, Ethics, Learning).

## 🚀 Quick Start

### Prerequisites
- Python 3.8+
- pip, virtualenv
- Docker (optional for containerized environment)

### Installation

```bash
# Clone this repository
git clone https://github.com/yourusername/automation-unlocked
cd automation-unlocked

# Create a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Run Experiments

```bash
# Core Experiment: PID vs RL Control
cd 2_The_Core/2.2_Core_Experiment_PID_vs_RL/
python main.py

# Applied Experiment: Perception-to-Actuation
cd ../../3_The_Applications/3.2_Applied_Experiments/perception_actuation/
python main.py

# Applied Experiment: RPA Demo
cd ../rpa_demo/
python main.py
```
